Product Rental Management System

A full-stack **Product Rental Management System** built using Spring Boot for the backend and a frontend application for user interaction.  
This application allows users to manage rental products efficiently.
 Project Overview
The Product Rental Management System helps in:
- Managing rental products
- Tracking availability
- Performing CRUD operations on products
- Supporting scalable backend APIs using Spring Boot
  Tech Stack
 Backend
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven

 Frontend
- HTML, CSS, JavaScript
- React (optional / if used)
 Database
- MySQL
 Features
- Add new rental products
- Update product details
- View all available products
- Delete products
- RESTful APIs
- Clean layered architecture (Controller, Service, Repository)
